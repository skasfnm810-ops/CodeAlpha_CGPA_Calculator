# 🎓 CGPA Calculator (C++)

## 📌 Description
This project is a CGPA Calculator developed in C++.  
It calculates GPA for the current semester and overall CGPA including previous semesters.

## 🚀 Features
- Supports grades (A, A-, B+, B, etc.)
- Takes credit hours input
- Calculates current GPA
- Calculates overall CGPA

## 💻 How to Run
1. Compile the code:
   g++ main.cpp -o cgpa
2. Run the program:
   ./cgpa
