# 🎓 CGPA Calculator (C++)

## **📌 Description**
This project is a CGPA Calculator developed in C++ as part of my programming practice and internship tasks.

The application is designed to help students calculate their academic performance efficiently. It takes input for current semester courses, including grades (A, A-, B+, B, etc.) and credit hours, and computes the GPA for the semester.

Additionally, the program allows users to enter data from previous semesters, such as GPA and total credit hours, and calculates the overall CGPA by combining all academic records.

This project demonstrates my understanding of core C++ concepts, including:
- Functions  
- Conditional statements  
- Loops  
- User input handling  
- Basic data processing  

The entire logic and implementation of this project have been developed independently.

---

## **🚀 Features**
- Supports grades (A, A-, B+, B, etc.)  
- Takes credit hours as input  
- Calculates current semester GPA  
- Calculates overall CGPA  
- Handles multiple semesters
